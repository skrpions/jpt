App Angular - Nodejs

Esta app muestra en el Frontend los valores registrados en la Base de datos de manera automática, además se grafican los datos de forma estática.

Se creó la tabla "índice ambiental" en la Bd y es posible calcular el resultado de la siguiente ecuación en tiempo real: índice ambiental = (0,5*Temperatura) + (0,15*Humedad) + (0,2*Nivel).

Condiciones:

1:. La Bd fue convertida de Sqlite a Mysql

2:. Los datos de la Bd no se gráfican de manera automática
